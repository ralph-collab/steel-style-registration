# Steel & Style Registration System

This repository contains the Steel & Style registration form and Google Apps Script backend.

## Files

- index.html = registration form
- logo.png = logo
- show-car.png = vehicle image
- apps-script/Code.gs = backend logic
- apps-script/appsscript.json = Apps Script config

## Required Script Properties (Apps Script)

- SQUARE_ACCESS_TOKEN
- SQUARE_LOCATION_ID
- ADMIN_EMAIL

## Notes

- Do not store secrets in this repo
- After updating Code.gs, redeploy Apps Script
